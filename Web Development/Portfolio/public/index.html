Vehicle TO Vehicle Communication(ADAS)

Advanced Driver Assistance System based on Vehicle-to-Vehicle (V2V) communication technology.

About

Advanced Driver Assistance System (ADAS) based on Vehicle-to-Vehicle (V2V) communication technology. This system provides a safer and smarter driving environment by enhancing connectivity and cooperation between vehicles on the road. It reduces accidents, improves traffic flow, and enhances the overall driving experience. The automation levels in this system are classified based on autonomous driving standards, ranging from full human control to complete automation, with a focus on improving emergency response and real-time road monitoring.

  • Improved Road Safety.
  • Enhanced Traffic Flow.
  • Faster Emergency Response.
  • Seamless Interaction Between Smart Vehicles.

In our project, we focus on developing a system that integrates ADAS and V2V technologies to provide innovative solutions such as Forward Collision Warning (FCW), Blind Spot Warning, Intersection Movement Assist (IMA),Don't Pass Warning (DNPW), Driver Monitoring System(DMS) and Traffic Sign Recognition(DMS) . These systems are implemented using a range of hardware components, including GPS sensors, ESP32 modules, display screens, distance-measuring sensors, and intelligent algorithms for real-time data processing and decision-making.

About System Archticture

This component diagram illustrates the infrastructure of the intelligent ADAS system based on Vehicle-to-Vehicle (V2V) communication technology. The system consists of:

Communication Interfaces:

  • HTTPS & Web Socket for server/cloud communication
  • ESP-NOW for ESP32 module communication
  • USB for sensor connections
  • SPI for display connections

Core Components:

  • Central processing unit (Raspberry Pi 4 Model B)
  • Raspberry Pi Camera v2 for computer vision
  • 10 DOF IMU sensor (MPU6050 + HM C5883L + BMP08) for motion measurement
  • ESP32 modules for motor control and user interface
  • Servo motor and LCD display
  • Firebase SDK for data storage and analytics

Control Modules:

  • ATMEGA32 for movement direction control (Left/Right)
  • Motor Driver unit
